## 🏛️ Project Overview
**FundedAI** is a high-performance autonomous factory for short-form video production. Unlike standard automation tools, FundedAI utilizes a **Caption-First Neural Pipeline** to calculate precision video durations based on text complexity, ensuring 100% watch-time retention through psychological timing.

## 🧠 Technical Architecture

### 1. Pre-Edit Neural Analysis
The pipeline begins by analyzing raw footage via **Gemini 1.5 Flash**. It extracts semantic context and generates viral hook/CTA pairs before a single frame is cut.

### 2. Strategy-Aware Dynamics
Video duration is calculated dynamically based on the selected production strategy:
- **Aggressive (2.6 wps):** High-tension, fast-paced loops.
- **Mysterious (2.1 wps):** Slower, curiosity-driven authority content.
- **Pattern Interrupt:** 5-6s shock-drops for maximum loop triggers.

### 3. Media Assembly
- **Engine:** FFmpeg (Baseline Profile Level 3.0)
- **Sanitization:** Auto-normalization of audio streams and pixel formats (YUV420P).
- **Hardening:** 300s render timeouts and ghost-watermark prevention.

## 🚀 Quick Start

### 1. Environment Setup
```bash
git clone https://github.com/Ninja-69/POST-SYSTEMS.git
pip install -r requirements.txt
```

### 2. Configuration
Create a `.env` file in the root directory:
```env
GEMINI_API_KEYS=your_key_1,your_key_2
TELEGRAM_BOT_TOKEN=your_token
MONGO_URI=your_mongodb_uri
```

### 3. Deploy Production
```bash
python main.py
```
Select `[1] PRODUCE` and choose a Strategy Archetype to begin autonomous batching.
